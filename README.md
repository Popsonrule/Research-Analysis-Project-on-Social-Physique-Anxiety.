# Research-Analysis-Project-on-Social-Physique-Anxiety.
This is the analysis of responses collected for a research project titled; "Influence of Social Physique Anxiety on Stress and Eating Habit among Students"
The raw data was collected with google form questionnaire and was appropriately cleaned, sorted, and transformed by MS Excel. It was then imported to SPSS for the generation of frequency tables and correlational analysis, which was used to test for the relationship between variables
